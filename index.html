LIMS-ELN Integration Platform

GitHub

Middleware platform designed for bidirectional data synchronization between Laboratory Information Management Systems and Electronic Laboratory Notebooks. Enables seamless data flow between laboratory equipment, LIMS databases, and researcher notebooks while maintaining complete audit trails for regulatory compliance.

Trained three machine learning models for data quality assurance: anomaly detection using Isolation Forest to identify unusual sample patterns, entity extraction with spaCy NER for automated metadata capture, and compliance scoring to ensure FDA 21 CFR Part 11 requirements. Built asynchronous data pipelines capable of processing over 1000 samples daily with complete audit trail documentation.

Regulatory Compliance Tracker

GitHub

Automated compliance monitoring system with OCR document processing and NLP analysis designed for pharmaceutical laboratories. The system continuously monitors regulatory documents, identifies compliance gaps, and tracks document expiration dates to ensure laboratories maintain regulatory standards across multiple frameworks.

Developed five machine learning models including document classifier achieving 94.5% accuracy using Random Forest, compliance scorer with Gradient Boosting for risk assessment, entity extractor for automated data capture from documents, anomaly detector for unusual compliance patterns, and text similarity analyzer for policy matching. Built real-time dashboard with gap analysis and document expiration tracking supporting FDA 21 CFR Part 11, GLP, and ISO 17025 standards.

Technologies: Python, Tesseract OCR, Scikit-learn, spaCy, Transformers, PostgreSQL, React
